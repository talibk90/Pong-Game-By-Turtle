# Pong Game - Python Turtle Graphics

A classic Pong game implementation using Python's `turtle` module. Challenge a friend in this local multiplayer game!

## Features

-   **Classic Gameplay**: Two-player paddle and ball mechanics.
-   **Scoreboard**: Tracks scores for both players.
-   **Sound Effects**: "Topple" sound plays when the ball hits a paddle.
-   **Start/Pause**: Press `Space` to pause and resume the game.
-   **Reset Game**: Press `R` to reset scores and ball position at any time.
-   **Win Condition**: First player to reach 10 points wins.
-   **Fair Gameplay**: Ball speed increases only on paddle hits, preventing impossible wall-bounce speeds.

## Controls

### Left Player
-   **W**: Move Up
-   **S**: Move Down

### Right Player
-   **Up Arrow**: Move Up
-   **Down Arrow**: Move Down

### Game Controls
-   **Space**: Start / Pause Game
-   **R**: Reset Game

## How to Run

1.  Ensure you have Python installed.
2.  Run the game:
    ```bash
    python main.py
    ```
3.  Enter player names when prompted.

## Requirements
-   Python 3.x
-   `turtle` (standard library)
-   `winsound` (standard library for Windows)
